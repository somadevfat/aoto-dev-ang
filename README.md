# 🚀 自動開発プロセス構築ガイド (Anti-QOL Project)

このリポジトリは、Antigravity の機能を最大限に活かして、要件定義書からアプリケーションを自動開発するための環境構築ガイドおよび関連ドキュメントを管理しています。

要件定義書のみを渡したら、人間によるセキュリティチェック・デプロイを除き、コードレビューなしで完成状態のアプリが出来上がる環境の構築を目指します。

## 📋 目次 / ドキュメント一覧 (auto-dev/)

自動開発プロセスの詳細は `auto-dev/` ディレクトリに分割して格納されています。
