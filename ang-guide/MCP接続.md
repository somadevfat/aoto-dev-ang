# MCP (Model Context Protocol) 統合ガイド

Antigravity エディタは、**Model Context Protocol (MCP)** をサポートしています。これにより、エディタをローカルツール、データベース、外部サービスと安全に接続し、AI にリアルタイムのコンテキストを提供することが可能になります。

---

## 💡 MCP とは？

MCP は、Antigravity と開発環境全体を繋ぐ**強力な架け橋**です。
通常、データベースのスキーマやログなどの情報を AI に伝えるには手動でのコピー＆ペーストが必要ですが、MCP を使用することで Antigravity が必要な情報を**直接、自動的に取得**できるようになります。

## 🔥 主な機能

### 1. コンテキストリソース

AI は接続された MCP サーバーからデータを読み取り、より精度の高い提案を行います。

- **SQL クエリの補助**: `Neon` や `Supabase` の live スキーマを直接参照し、正確なテーブル名やカラム名を提案します。
- **デバッグの効率化**: `Netlify` や `Heroku` から最新のビルドログを直接引き出し、原因特定を迅速化します。

### 2. カスタムツール

MCP を介して、Antigravity から外部サービスへのアクションを実行できます。

- **連携例**: 「この TODO を `Linear` のチケットにして」や「`Notion` で認証パターンを検索して」といった指示が可能です。

---

## ⚙️ 接続方法

MCP サーバーとの接続管理は、エディタに組み込まれた **MCP Store** で簡単に行えます。

1. **ストアを開く**: サイドパネル上部の `...` ドロップダウンから **MCP Store** を選択します。
2. **インストール**: リストから必要なサーバーを選択し、`Install` をクリックします。
3. **認証**: 画面の指示に従い、アカウントを安全に連携させます。

### 🛠️ カスタム MCP サーバーの追加

独自の MCP サーバーを接続する場合の手順：

1. MCP ストアのパネルから **Manage MCP Servers** をクリック。
2. **View raw config** を選択。
3. `mcp_config.json` を開き、サーバー設定を記述・保存します。
