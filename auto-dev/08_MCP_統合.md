## 8. MCP 統合

### 8.1 開発で活用すべきMCPサーバー

| サービス            | 用途                              |
| :------------------ | :-------------------------------- |
| **GitHub**          | Issue管理、PR作成、コードレビュー |
| **Supabase / Neon** | DB スキーマの直接参照、クエリ補助 |
| **Linear**          | タスク管理連携（TODO→チケット化） |
| **Firebase**        | デプロイ連携                      |

### 8.2 設定方法

1. サイドパネルの `...` → **MCP Store** を開く
2. 必要なサーバーを **Install**
3. 認証を完了する

> カスタムMCPサーバーが必要な場合は `mcp_config.json` を直接編集する。
